## Turn Claude Code from one generic assistant into a team of specialists

gstack provides eight opinionated workflow skills for Claude Code. Plan review, code review, one-command shipping, browser automation, QA testing, and engineering retrospectives — all as slash commands.

### Without gstack

- The agent takes your request literally — it never asks if you're building the right thing
- "Review my PR" gives inconsistent depth every time
- "Ship this" turns into a long back-and-forth about what to do
- The agent can write code but can't see your app — it's half blind
- You still do QA by hand: open browser, click around, check pages, squint at layouts

### With gstack

Switch cognitive modes on demand — founder taste, engineering rigor, paranoid review, or fast execution.

## Key features

- **Eight opinionated workflow skills** as slash commands for different cognitive modes
- **Persistent headless browser** with sub-second latency and session state
- **Ref-based element interaction** using accessibility tree — no CSS selectors
- **Diff-aware QA testing** that analyzes your git changes and tests affected pages
- **Cookie import from real browsers** (Chrome, Arc, Brave, Edge, Comet)
- **Greptile PR review integration** with automatic triage
- **Multi-workspace support** with isolated browser instances
- **Engineering retrospectives** with team-aware metrics and feedback
